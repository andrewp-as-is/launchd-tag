### Installation
```bash
$ [sudo] pip install launchd-tag
```

```bash
$ [sudo] npm i -g launchd-tag
```

#### How it works
color|description
-|-
none|running
Gray|not running
Red|non-zero exit status
Orange|not empty stderr (`StandardErrorPath`)

#### Requirements
```bash
$ brew install tag
```

